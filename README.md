# Stock Watchlist Dashboard

A demo React dashboard for tracking stocks, indices, F&O, mutual funds, and more. Includes search, tabs, profile management, and modern UI components.

---

## Features

- Watchlist with multiple tabs: Stocks, Indices, F&O, Mutual Funds
- Search bar to filter stocks
- Orders, Portfolio, and Bids sections with interactive tabs
- Profile page with privacy mode toggle
- Stylish scrollbars and icons
- Responsive and modern UI with Tailwind CSS

---

## Tech Stack

- React (v18+)
- Tailwind CSS
- Lucide Icons
- React Router DOM

---

## Installation

### 1. Clone the repository

### 2. Install dependencies

```bash
git clone https://github.com/kprateek87/zerodhapanel.git
```

```bash
npm install
```

or if you use yarn:

```bash
yarn
```

---

## Running the Project

Start the development server:

```bash
npm start
```

or

```bash
yarn start
```

## Usage

- Navigate between tabs to view stocks, indices, F&O, and mutual funds.
- Use the search bar to filter stocks by ticker or name.
- Toggle Privacy Mode on the profile page.
- Scrollbars in watchlists are customized for a modern feel.
